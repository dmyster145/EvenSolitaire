# Packaging and deployment (G2)

- **Build:** `npm run build` — output in `dist/`.
- **Run locally:** `npm run dev` — open the dev URL in Even App (or scan QR with `npx evenhub qr` if available).
- **Deploy:** Host `dist/` on any static host; open the deployed URL in the Even App to use the widget on G2.
- **CLI:** See [SDKandDevResources.md](SDKandDevResources.md) for `@evenrealities/evenhub-cli` and `evenhub pack` usage.
