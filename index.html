Getting Started

Welcome

EvenSolitaire is Klondike Solitaire built for Even Realities G2 gesture input: scroll, tap, and double-tap.

Launch

On launch, the app restores your last saved game when available. If no valid save is found, it deals a new shuffled game.

  • Restores game state when a save exists
  • Restores your Move Assist setting with the save
  • After a win, tap starts a new game; double-tap opens the menu

What This Covers

This page documents the current user-facing behavior: controls, menus, prompts, save/resume behavior, and the Klondike rules used by this build.

Board Layout

Core Piles

  • Stock: Face-down draw pile. Tap to draw three cards.
  • Waste: Face-up draw pile. Only the top card is playable.
  • Foundations (4): Build each suit from Ace to King.
  • Tableau (7): Main play columns built downward in alternating colors.

Focus Navigation

The app is gesture-driven, so a focus cursor moves between pile targets:

  • Browse mode skips empty non-stock piles
  • Browse mode also skips foundations
  • Destination mode moves across tableau and foundation drop targets
  • With Move Assist ON, illegal destinations are skipped
Note: Foundations can be focused during placement, even though browse mode skips them.
Controls

Use three gestures on the G2 touchpad:

Scroll Move focus between piles, destinations, or menu items.
Tap Draw from stock, select a source pile, place cards, or confirm a menu option.
Double Tap Open or close the menu, or cancel an active selection. On win, open the menu.

Gameplay Flow

Basic Move Flow

  1. Scroll to the pile you want to use (stock, waste, or tableau)
  2. Tap to select the source (or draw from stock)
  3. With Move Assist ON, if an auto-destination is available, focus jumps there. Waste prefers the leftmost legal tableau (then foundation). Tableau only auto-jumps when the face-up run has exactly one legal destination.
  4. Tap to place

Selecting Tableau Runs (Multi-card Moves)

In destination mode, tapping the same tableau column cycles the carried run size (1, 2, 3, and so on up to the visible count, then back to 1).

This is how the app supports moving tableau runs without an extra gesture.

Invalid Moves and Canceling

  • Illegal placements blink briefly and return focus to the source
  • If a selected source has no legal moves, the HUD may show 0 Legal Moves
  • Double-tap cancels the current selection and returns to browsing
Rules (Klondike Solitaire)

Official Rules Reference

Baseline reference: Bicycle Cards' standard Klondike Solitaire rules. EvenSolitaire follows that structure, with G2-specific interaction notes below.

Objective

Move all cards to the four foundations, building each suit from Ace to King.

Initial Deal (Implemented)

  • Seven tableau piles are dealt with 1 through 7 cards
  • Only the top card in each tableau pile starts face-up
  • The remaining cards form the stock

Foundation Rules

  • Start with an Ace
  • Build upward in the same suit (A to K)

Tableau Rules

  • Build downward in rank with alternating colors
  • Move valid face-up runs between tableau piles
  • Only a King (or a run starting with a King) can move to an empty tableau column
  • When the top face-down card is uncovered, it is flipped face-up automatically

App-Specific Clarifications

  • Board draw: Tapping the stock draws three cards (or fewer when fewer cards remain)
  • Menu draw assist: The menu’s Draw Card option draws one card to help when you are stuck
  • Auto-destination priority (Move Assist ON): Selecting from waste auto-focuses the leftmost legal tableau destination, then foundation if needed. Selecting from tableau auto-focuses only when the face-up run has exactly one legal destination. The move is not placed until you tap.
  • Recycle: The waste can be recycled when the stock is empty
  • Win condition: You win when all four foundations have 13 cards
Tip: Turn on Move Assist to skip illegal destinations while learning the rules.
Menu Options

Double-tap during normal play to open the menu. Scroll to browse. Tap to select. Double-tap again to close the menu.

Move Assist

Toggles destination filtering and legal move counts in the info panel.

  • On: Destination scrolling skips illegal drops while carrying cards
  • Off: You can scroll to any tableau or foundation destination

Draw Card

Draws one card from the stock as an assist action when you are stuck. This is separate from the board stock draw, which draws three cards.

Reset

Shows a Yes/No confirmation before starting a new game.

Exit

Closes the app and saves the current game state before exit. Use double-tap if you only want to close the menu.

Status & Feedback

HUD Prompts

The HUD changes by mode and may show prompts such as:

  • Select source pile
  • Focus: Stock, Focus: Waste, Focus: T1 ... Focus: T7
  • [Menu] plus the current menu items
  • You win! Tap for new game
  • Status Message and other short messages

Info Panel / Focus Details

  • Shows cards in the focused pile (or the active source pile while carrying cards)
  • Shows Move Assist: ON/OFF
  • When Move Assist is ON, shows a legal move count for the focused pile

Menu in HUD

The menu renders in the HUD panel with the selected line highlighted. Reset switches the HUD menu to a confirmation prompt.

Save & Resume

Autosave

The app autosaves your game state and Move Assist setting. Saves are queued and debounced so rendering stays responsive under load.

Persistence

  • Even Hub / G2: Uses the bridge storage API
  • Browser fallback: Uses localStorage

Resume on Launch

On startup, the app restores a saved game and Move Assist before creating the initial state when valid save data exists.

Debug Console (Perf Panel)

Panel Controls

When It Appears

The panel is hidden by default and appears when DOM perf logging is enabled in src/perf/log.ts (for example, PERF_LOG_DOM_ENABLED).

Tips & Notes

Playing Efficiently

  • Use Move Assist while learning. It reduces destination scrolling noise.
  • If a tableau selection looks wrong, double-tap to cancel and reselect.
  • Tapping the same tableau column while carrying cards changes the run size.
  • Watch for newly exposed tableau cards. The app flips them automatically.
