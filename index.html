Getting Started

Welcome

EvenSolitaire is classic Klondike tuned for Even Realities G2: scroll, tap, and double-tap. Tiny gesture set, big card energy.

Launch

On launch, the app restores your latest valid save. If there is no save, it deals a fresh shuffle.

  • Restores game state when a save exists
  • Restores your Move Assist setting with the save
  • After a win, tap starts a new game; double-tap opens the menu

What This Covers

This page mirrors current in-game behavior: controls, menus, prompts, save/resume, and Klondike rules as implemented in this build.

Board Layout

Core Piles

  • Stock: Face-down draw pile. Tap to draw three cards.
  • Waste: Face-up draw pile. Only the top card is playable.
  • Foundations (4): Build each suit from Ace to King.
  • Tableau (7): Main play columns, built downward in alternating colors.

Focus Navigation

Focus is your cursor. It moves by mode:

  • Browse mode skips empty non-stock piles and skips foundations
  • Destination mode moves across tableau and foundation drop targets
  • With Move Assist ON, illegal destinations are skipped
Note: Foundations are skipped during browse mode, but are available during placement.
Controls

Three gestures run the whole table:

Scroll Navigate focus between piles, destinations, and menu items.
Tap Select, place, draw, or confirm. Context decides.
Double Tap Open/close menu, or cancel an active selection. On win, it opens menu.

Gameplay Flow

Basic Move Flow

  1. Scroll to the source pile (stock, waste, or tableau)
  2. Tap to draw or pick up cards
  3. With Move Assist ON, focus may auto-jump to a legal destination (waste prefers foundation, then leftmost legal tableau; tableau jumps only with exactly one legal destination), and tapping a tableau Ace auto-places it to foundation
  4. Tap again to place

Selecting Tableau Runs (Multi-card Moves)

In destination mode, tapping the same tableau column cycles carried run size (1, 2, 3, ... up to visible count, then back to 1).

Think of it as a built-in run-size dial, no extra gesture required.

Invalid Moves and Canceling

  • Illegal placements blink briefly, then focus snaps back to source
  • If a selected source has no legal moves, the HUD may show 0 Legal Moves
  • Double-tap cancels the current selection and returns to browsing
Rules (Klondike Solitaire)

Official Rules Reference

Baseline reference: Standard Klondike. EvenSolitaire keeps that rule core and adapts interaction for G2 gestures.

Objective

Move all cards to the four foundations, building each suit from Ace to King.

Initial Deal (Implemented)

  • Seven tableau piles are dealt with 1 through 7 cards
  • Only the top card in each tableau pile starts face-up
  • The remaining cards form the stock

Foundation Rules

  • Start with an Ace
  • Build upward in the same suit (A to K)

Tableau Rules

  • Build downward in rank with alternating colors
  • Move valid face-up runs between tableau piles
  • Only a King (or a run starting with a King) can move to an empty tableau column
  • When the top face-down card is uncovered, it is flipped face-up automatically

App-Specific Clarifications

  • Board draw: Tapping the stock draws three cards (or fewer when fewer cards remain)
  • Menu draw assist: The menu’s Draw Card option draws one card (and handles recycle first when needed)
  • Auto-destination priority (Move Assist ON): Selecting from waste auto-focuses foundation first, then the leftmost legal tableau destination if no legal foundation exists. Selecting from tableau auto-focuses only when the face-up run has exactly one legal destination, except tableau Aces, which auto-place to foundation on tap even if other moves exist.
  • Recycle: The waste can be recycled when the stock is empty
  • Win condition: You win when all four foundations have 13 cards
Tip: Turn on Move Assist to skip illegal destinations while learning the rules.
Menu Options

Double-tap in normal play to open menu. Scroll to choose, tap to confirm, double-tap to close.

Move Assist

Toggles destination filtering and legal-move counts in the info panel.

  • On: Destination scrolling skips illegal drops while carrying cards
  • Off: You can scroll to any tableau or foundation destination

Draw Card

Draws one card from stock as an assist action. This is separate from board stock draw, which draws three.

Reset

Shows a Yes/No confirmation before starting a new game.

Exit

Saves current game state, then exits the app. Use double-tap if you only want to close menu.

Status & Feedback

HUD Prompts

The HUD changes by mode. Common lines include:

  • Select source pile
  • Focus: Stock, Focus: Waste, Focus: T1 ... Focus: T7
  • MENU with the selected option marked by >
  • You win! Tap for new game
  • Short state messages such as 0 Legal Moves

Info Panel / Focus Details

  • Shows cards in the focused pile (or the active source pile while carrying cards)
  • Shows Move Assist: ON/OFF
  • When Move Assist is ON, shows a legal move count for the focused pile

Menu in HUD

The menu lives in the HUD panel. Reset swaps options to a Yes/No confirmation prompt.

Save & Resume

Autosave

The app autosaves game state and Move Assist. Saves are queued and debounced so gameplay stays responsive under load.

Persistence

  • Even Hub / G2: Uses the bridge storage API
  • Browser fallback: Uses localStorage

Resume on Launch

On startup, valid save data restores both game state and Move Assist before initial state is finalized.

Debug Console (Perf Panel)

The floating perf console panel is preserved on this page and uses the same DOM IDs and controls as the original Solitaire index.html.

Panel Controls

  • Hide / Show

  • Download

When It Appears

The panel is hidden by default and appears when DOM perf logging is enabled in src/perf/log.ts (for example, PERF_LOG_DOM_ENABLED).

Tips & Notes

Playing Efficiently

  • Use Move Assist while learning. It cuts destination-scrolling noise.
  • If a tableau selection looks wrong, double-tap to cancel and reselect.
  • Tapping the same tableau column while carrying cards changes the run size.
  • Watch for newly exposed tableau cards. The app flips them automatically.
